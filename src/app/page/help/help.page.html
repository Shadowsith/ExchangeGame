Help To use this app follow the instructions 1. Open alphavantage.co 2. Claim your api key for free 3. Save it to a secure location 4. Go to the app settings 5. Add your api key at "Stock API Key" 6. Save the settings You are now able to use the stock market functions of the app. Have fun! The Alpha Vantage Stock API is free for non-commercial usage. For more information read their terms of service. Did you find a bug? Please write an issue at the GitHub project site.
